══════════════════════════════════════════════════════════════════════
THE CORE PHILOSOPHY
══════════════════════════════════════════════════════════════════════

  You are MEANT to die.

  Each run, your squad ascends The Lift — a massive space elevator
  infested with Raptures. You fight floor by floor, earning skill
  points (spent during this run) and Pristine Cores (persistent
  currency). You push as high as you can. Your squad will die.
  Then you spend your cores, get stronger, and climb higher next
  time.

  The game is not about winning a single run. It's about how far
  you can push before falling — and how much further you get on
  the next attempt.

  ┌─────────────────────────────────────────────────────────┐
  │                                                         │
  │         DIE → UPGRADE → CLIMB HIGHER → DIE AGAIN       │
  │         ↳ Each loop makes you a little stronger         │
  │                                                         │
  └─────────────────────────────────────────────────────────┘


══════════════════════════════════════════════════════════════════════
THE LIFT — STRUCTURE
══════════════════════════════════════════════════════════════════════

  The Lift is INFINITE (or has a very high cap like 100+ floors).
  It is divided into SECTORS of 20 floors each.

  ┌─────────────────────────────────────────────────────────┐
  │  SECTOR 5: THE THRONE (81-100)                          │
  │    └─ Biome: Queen's chamber, golden corruption         │
  │                                                         │
  │  SECTOR 4: QUEEN'S DOMAIN (61-80)                       │
  │    └─ Biome: Dark Gothic, red lighting, elite-heavy     │
  │                                                         │
  │  SECTOR 3: THE MAW (41-60)                              │
  │    └─ Biome: Organic, alien, flesh-like walls           │
  │                                                         │
  │  SECTOR 2: MID SHAFT (21-40)                            │
  │    └─ Biome: Corrupted industrial, rust, vines          │
  │                                                         │
  │  SECTOR 1: LOWER SHAFT (1-20)                           │
  │    └─ Biome: Cold steel, industrial, standard enemies   │
  │                    ↕                                    │
  │              ENTRY POINT                                │
  └─────────────────────────────────────────────────────────┘

  EACH SECTOR HAS:
  ─ A unique biome/visual theme (environment, color palette,
    enemy variations, hazards)
  ─ 5x mini-boss floors (5, 10, 15 — plus the Sector Boss at 20)
  ─ Increasing difficulty that ends with a SECTOR BOSS
    that tests everything you've learned

  NEW PLAYER EXPERIENCE (First few runs):
  ─ Die around floor 8-15. Earn ~6-13 SP + 0-2 cores.
  ─ Spend cores on Starting HP upgrade. Next run push to floor 12-18.
  ─ Keep climbing. First Sector Boss clear is a MAJOR milestone.
  ─ Each full sector clear unlocks a CHECKPOINT for future runs.


══════════════════════════════════════════════════════════════════════
FLOOR DESIGN
══════════════════════════════════════════════════════════════════════

  Each floor is a SELF-CONTAINED ARENA (~3,000-4,000 units).
  No walking. No exploration. Drop in. Fight. Clear. Ascend.

  FLOOR TYPES:

  ┌──────────┬────────────────────────────┬─────────────────┐
  │ TYPE     │ WHAT HAPPENS               │ APPEARS         │
  ├──────────┼────────────────────────────┼─────────────────┤
  │ COMBAT   │ Standard enemy wave.       │ Default type    │
  │          │ Kill all to clear.         │ (any floor)     │
  ├──────────┼────────────────────────────┼─────────────────┤
  │ AMBUSH   │ Starts empty. Enemies      │ Floors 3+       │
  │          │ drop in from all sides     │ (occasional)    │
  │          │ after 3 seconds.           │                 │
  ├──────────┼────────────────────────────┼─────────────────┤
  │ DEFENSE  │ Protect a point for 60s.   │ Floors 5+       │
  │          │ Enemies swarm the point.   │ (rare)          │
  ├──────────┼────────────────────────────┼─────────────────┤
  │ GAUNTLET │ 3 consecutive waves with   │ Floors 10+      │
  │          │ no break between them.     │ (rare)          │
  │          │ High intensity.            │                 │
  ├──────────┼────────────────────────────┼─────────────────┤
  │ BOSS     │ Single super-boss enemy.   │ Floors 5, 10,   │
  │          │ No trash mobs (or very     │ 15 (mini-boss), │
  │          │ few). Pure boss fight.     │ 20 (sector boss)│
  └──────────┴────────────────────────────┴─────────────────┘

  FLOOR DIFFICULTY SCALING:
  ─ Enemy HP:   +12% per floor (compounding)
  ─ Enemy ATK:  +8% per floor (compounding)
  ─ Enemy count: +2 per floor (linear)
  ─ New types:  Tanks at floor 4+, Exploders at 6+,
                Shielders at 8+, Elites at 10+
  ─ Hazards:    Floor hazards start appearing at sector 2+
  ─ Enrage:     At floor 10+ within a sector, if you take
                more than 3 minutes on a floor, enemies get
                +25% speed and +25% attack speed

  The goal: difficulty ramps aggressively so you HIT A WALL.
  Your first run you breeze through floors 1-5, struggle on
  6-10, and die around 12-15. Each subsequent run you push
  1-3 floors further.

  TIME PER FLOOR:
  ─ Combat floors: 1.5-3 minutes
  ─ Boss floors:    2-4 minutes
  ─ Full sector (20 floors): ~40-60 minutes
  ─ Typical first death:      run ends ~15-25 minutes in

══════════════════════════════════════════════════════════════════════
DEATH & REVIVAL
══════════════════════════════════════════════════════════════════════

  WHEN A CHARACTER HITS 0 HP ON A FLOOR:
  ─ They are DOWNED. They fall. They can't move or attack.
  ─ You finish the floor without them.
  ─ After the floor ends, they are DEAD for the rest of the
    run — UNLESS you have a Spare Body item.

  SPARE BODY ITEM:
  ─ Acquired by choosing it as a floor reward bonus
  ─ Stored in your inventory (visible on the reward screen)
  ─ Can be SAVED — you don't have to use it immediately
  ─ After clearing a floor, if you have a Spare Body,
    a "Revive" button appears next to dead characters
  ─ Using it revives 1 dead character with 25% HP
  ─ The item is consumed on use
  ─ You can hold up to 3 Spare Bodies at once

  RUN ENDS WHEN: All 3 squad members are dead simultaneously.
  ─ If you finish a floor with 2 dead and 1 alive, the run
    continues. Next floor you fight with just 1 character
    (you control them, no AI allies).
  ─ If that last character dies on the next floor, run over.

  STRATEGIC IMPLICATIONS:
  ─ Sacrificing a weak character to preserve the strong ones
    is a valid strategy
  ─ Going into a boss floor with a dead character is risky
    but might be worth it if the rewards are good
  ─ Spare Bodies are PREMIUM rewards — choosing them over
    a damage boost is a real trade-off


══════════════════════════════════════════════════════════════════════
SKILL POINTS & TALENT TREE (Run-Only)
══════════════════════════════════════════════════════════════════════

  Skill points are RUN-ONLY. They reset to 0 when the run ends.
  This is the core of the roguelite loop — each run you rebuild
  your characters from scratch.

  HOW IT WORKS:
  ─ Each floor clear gives 1 skill point (shown on reward screen)
  ─ During a run, press TAB to open the Talent Tree (same UI)
  ─ Spend skill points on the current 3 characters' talent trees
  ─ You can distribute points however you want between the 3
    characters
  ─ Talent tree is IDENTICAL to what exists now — unlock burst,
    special, passives per character

  TYPICAL SKILL POINT BUDGET PER RUN:
  ─ Die on floor 10:    10 SP (barely 1 char's burst)
  ─ Die on floor 18:    13 SP (1 burst + 1 special)
  ─ Clear sector 1 (20): 16 SP (2 chars with bursts)
  ─ Clear sector 2 (40): 28 SP (full squad built out)

  This means a run that ends early gives you barely enough
  to unlock 1 character's burst. A deep run lets you build
  out multiple characters with passives and upgrades.

  STRATEGY:
  ─ Do you dump all points into 1 character (strong carry)?
  ─ Or spread them evenly (balanced squad, less depth)?
  ─ Talent choices change based on which floor bonuses you
    picked — damage-focused build vs survival-focused build

══════════════════════════════════════════════════════════════════════
CHECKPOINT SYSTEM
══════════════════════════════════════════════════════════════════════

  Every 20 floors = 1 CHECKPOINT unlocked at the COMMAND CENTER.

  UNLOCKING A CHECKPOINT:
  ─ Defeat the Sector Boss on floor 20 → "SECTOR 1 CLEARED"
  ─ Command Center now shows "START FROM SECTOR 2 (Floor 21)"
  ─ Same for floor 40, 60, 80, 100...

  STARTING FROM A CHECKPOINT:
  ─ Before a run, choose: START FROM FLOOR 1 or START FROM
    the highest unlocked checkpoint
  ─ No bonus rewards. No catch-up cores. The challenge IS
    the reward — you're playing the hard part because you
    proved you can handle it.
  ─ Bragging rights: "I start from floor 41 now" is the flex

  WHY START FROM A HIGHER FLOOR?
  ─ Shorter path to the next checkpoint (20 floors vs 40+)
  ─ More efficient core farming (bosses are denser)
  ─ Skill points come faster relative to time spent
  ─ You've already "beaten" the early sectors — replaying
    them is just a time tax

  "FIRST CLEAR" STATUS:
  ─ The game tracks your HIGHEST FLOOR ever reached
  ─ Displayed prominently on the main menu / start screen
  ─ "HIGHEST: FLOOR 37 | CHECKPOINTS: SECTOR 1"
  ─ This IS your progression bar. Each number is a trophy.

  EXAMPLE PROGRESSION ARC:
  Run  1: Die on floor 12.  (Highest: 12)
  Run  2: Die on floor 16.  (Highest: 16)
  Run  3: Die on floor 19.  (Highest: 19)
  Run  4: CLEAR SECTOR 1!   (Highest: 20, Checkpoint: Sector 2)
  Run  5: Start at 21, die on floor 26. (Highest: 26)
  Run  6: Start at 21, die on floor 32. (Highest: 32)
  Run  7: Start at 21, die on floor 38. (Highest: 38)
  ...

  Each run takes ~15-40 minutes depending on how far you get.
  The climb is real. Every floor past your previous best feels
  like an achievement.

  EXTRACT: Safe. Keep everything. Unlock the checkpoint.
  ─ Good for: first time clearing a sector, low squad HP,
    no Spare Bodies left, want to bank cores safely.

  CONTINUE: Risky. If you die in the next sector, you still
  keep the checkpoint from the sector you cleared. But you
  lose any unspent skill points and unbanked rewards.
  ─ Good for: strong squad, full HP, multiple Spare Bodies,
    want to push for the next checkpoint.

  This creates a risk/reward decision at every sector boundary.

SECTOR & BIOME THEMES
══════════════════════════════════════════════════════════════════════

  SECTOR 1 — LOWER SHAFT (Floors 1-20)
  ─ Visual: Cold steel, industrial, fluorescent lights, pipes
  ─ Enemies: Standard raptures only (basic, tank at 4+)
  ─ Hazards: None
  ─ Boss: Corrupted Security System (mechanical boss)
  ─ Vibe: "This is the tutorial. Learn the game."

  SECTOR 2 — MID SHAFT (Floors 21-40)
  ─ Visual: Rusted metal, broken lights, vines, water leaks
  ─ Enemies: All standard + exploders, first elites at 30+
  ─ Hazards: Floor 28+ — sparking wires (damage if touched)
  ─ Boss: Corrupted Commander (elite Nikke, humanoid)
  ─ Vibe: "The elevator has been abandoned for a long time."

  SECTOR 3 — THE MAW (Floors 41-60)
  ─ Visual: Organic growth, flesh-like walls, bioluminescent
  ─ Enemies: Heavily corrupted variants, elite-heavy
  ─ Hazards: Floor 45+ — acid pools, moving crusher walls
  ─ Boss: Rapture Breeder (biological horror)
  ─ Vibe: "The Raptures have made this their nest."

  SECTOR 4 — QUEEN'S DOMAIN (Floors 61-80)
  ─ Visual: Dark Gothic, red and gold, religious iconography
  ─ Enemies: All elite variants, new "Cultist" type
  ─ Hazards: Floor 65+ — flame vents, collapsing platforms
  ─ Boss: Queen's Herald (first direct servant of N01)
  ─ Vibe: "You are in her territory now. She knows you're here."

  SECTOR 5 — THE THRONE (Floors 81-100)
  ─ Visual: Golden corruption, impossible geometry, floating
  ─ Enemies: All types maxed, including new "Angel" variants
  ─ Hazards: Every floor — reality-warping patterns
  ─ Boss: N01 — The Rapture Queen (final challenge)
  ─ Vibe: "This is the end. She is waiting."

  Beyond 100: "ENDLESS MODE" unlocked. Random biome rotation.
  No more checkpoints. Pure leaderboard chasing.
