Talent Tree — A2 Frame · Arrangements · Swappable Tabs

click ATTACK / SKILL / BURST on any mockup to swap trees · five arrangements, sparse → dense
All keep the approved A2 header-band frame. The tabs now work — click them. Each tab shows that ability's own tree (the old single tree, now split across ATTACK / SKILL / BURST). What changes between the five sections is the arrangement of talents inside, since the count per tree is still open. B1 is what we have, vertically centered (top padding = bottom). Talent names are placeholders to show the swap; geometry is the point.
