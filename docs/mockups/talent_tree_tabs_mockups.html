Talent Tree — 3-Tab Options · Attack / Skill / Burst

picking the TAB-SWAP mechanism only · all four show the identical state so only the chrome differs
Each character is moving from one tree to three trees, each its own tab: ATTACK (normal/left-click), SKILL (right-click special), BURST (ultimate). Skill points are shared and scarce, so players mix & match across all three — WoW-style. Every mockup below shows the same state on purpose (Snow White, the SKILL tab active, the same Auto-Turret tree, 3 SP unspent, ATTACK 2 / SKILL 4 / BURST 0 invested) so the only thing changing is how the tabs look and where they sit. Nodes/edges/panels reuse the approved C3a "Ability Lanes" vocabulary. Each tab surfaces its own invested-point count so the build distribution is legible at a glance.
