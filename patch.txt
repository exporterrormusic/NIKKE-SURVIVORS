Patch v0.2A  -  "CHRISTMAS EVE LAUNCH"
COMMANDER IMPROVEMENTS:
- Assault rifle bullets now have slight homing toward nearby enemies
- Assault rifle, sniper, and shotgun bullets have larger, more forgiving hitboxes
- "Good Genes" wave-end heal now triggers correctly

ANTI-CHEAT SYSTEM:
- Using cheats or debug menu during a match now taints the run
- Tainted matches are NOT recorded to leaderboards
- Tainted matches CANNOT unlock achievements
- Flag resets on starting a new run

NEW ACHIEVEMENTS:
- "Look Commander, No Hands!" - Defeat Wave 11 without firing any shots
  (Use summons, drones, mind control, etc. to win)

NEW SHOP UPGRADES:
- Kilo: "Filling Crown's Ball Pit" (20 cores)
  - +50% core drop chance on wave rewards
  - Bosses have 50% chance to drop an extra core
  - Passive: Active when Kilo is in squad with upgrade purchased

Patch v0.1K  -  "GODDESS FALL"
GODDESS FALL MODE:
- Rapture Queen HP increased to 6666 (scales with difficulty)
- Rapture Queen shield now 10% of max HP (666 base)
- Horde Summon: At 30% HP, summons 70 enemies from all map edges
- Core Overload: At 10% HP, triggers 10-second self-destruct countdown with 80% damage reduction
- Warning banners display for both events with smart text scaling
- Players start with 3 skill points in Goddess Fall mode
- Aggression system: Queen speeds up and attacks faster as HP drops
- Passive regeneration: 0.5%/s (high HP), 1%/s (mid), 2%/s (low HP)

- RAPUNZEL BURST: Fixed severe lag spike on launch. Restored the epic 0.5x camera zoom!

- ROCKETS: Optimized physics and texture generation for smoother flight.
NEW FEATURES:
- DAMAGE LOG: Added a detailed log of recent damage taken to the Pause and Defeat menus.
BUG FIXES:
- Fixed crash related to enemy viewport checks.
